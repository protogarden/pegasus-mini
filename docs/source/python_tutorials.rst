################
Python Tutorials
################

The Python Tutorials presented make use of the roboclaw.py python script which is imported as a module in the tutorials. The roboclaw python script can be found in the `Roboclaw Python Library`_. Inorder to import the module during the tutorials, the roboclaw.py script need to be in the same directory to where the tutorial scripts or commands are run. 

1. First navigate in terminal to where you would like to perform your Tutorials. The roboclaw.py file can be obtained by cloning the following repository: 

.. code-block:: bash

   git clone https://github.com/basicmicro/roboclaw_python_library.git
   
2. Next navigate into the roboclaw_python directory and run your commands or scripts within that directory.
